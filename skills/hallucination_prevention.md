# Accuracy & Hallucination Prevention

- If you are unsure about a function signature, library API, or file path, do not guess. Ask me to provide the relevant snippet or documentation.
- Do not invent file paths. If you need to create a new file, state the exact path you intend to use and confirm with me first.
- When using third-party libraries, assume the latest stable version. If you suggest a method that is deprecated, I will correct you.
- Prefer native browser APIs over external libraries unless the task explicitly requires a library.
